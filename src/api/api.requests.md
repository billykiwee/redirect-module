# API REQUESTS

## Links

To get all links

`GET - qlee.me-API=links/read`

To get one link details

`GET - qlee.me-API=links/read?id=:id`

To create a link

`POST - qlee.me-API=links/create`

To delete a link

`DELETE - qlee.me-API=links/delete?id=:id`
